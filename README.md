# Landing Page de Portfólio

Este é um projeto de landing page pessoal desenvolvido para apresentar o portfólio de Domingos Muratori, um desenvolvedor Front-End. O site foi criado com foco em design moderno, responsividade e interatividade, utilizando tecnologias web básicas para demonstrar habilidades em desenvolvimento front-end.

## Estrutura do Projeto

O projeto possui uma estrutura simples e organizada:

- `index.html`: Arquivo principal contendo a estrutura HTML da página
- `styles.css`: Arquivo de estilos CSS responsável pelo design e layout
- `imgs/`: Pasta contendo as imagens utilizadas no site (foto de perfil e capturas de projetos)
- `README.md`: Este arquivo de documentação
- `LICENSE`: Arquivo de licença do projeto

## Conceito

A landing page foi concebida como um portfólio digital que combina elegância visual com funcionalidade prática. O design adota um tema escuro com elementos de vidro (glassmorphism), criando uma estética moderna e profissional. A página inclui:

- **Navegação Fixa**: Menu superior com links âncora para as seções da página
- **Cabeçalho**: Apresentação pessoal com foto de perfil e título profissional
- **Seção Sobre**: Breve descrição das habilidades e paixão pelo desenvolvimento
- **Seção Projetos**: Galeria de projetos em formato de cards com links externos
- **Seção Contato**: Formulário integrado com WhatsApp para facilitar o contato

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página
- **CSS3**: Estilização avançada incluindo:
  - Variáveis CSS para cores consistentes
  - Flexbox e Grid para layouts responsivos
  - Animações e transições suaves
  - Efeitos de vidro (backdrop-filter)
  - Design responsivo para diferentes dispositivos
- **JavaScript**: Funcionalidade básica para o formulário de contato que abre o WhatsApp

## Funcionalidades

- Design totalmente responsivo
- Navegação suave entre seções
- Efeitos visuais modernos (partículas de fundo, hover effects)
- Integração direta com WhatsApp via formulário
- Links para projetos externos hospedados no GitHub Pages

## Como Visualizar

Para visualizar o projeto localmente:

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador web
3. Navegue pelas seções usando o menu ou scroll

O site é estático e não requer servidor para funcionar, sendo ideal para hospedagem em plataformas como GitHub Pages, Netlify ou Vercel.
